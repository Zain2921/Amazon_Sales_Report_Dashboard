# 📊 Amazon Sales Report Dashboard

A Power BI dashboard providing a comprehensive visual analysis of Amazon sales data. Built using **Microsoft Power BI** (Cloud), this report helps stakeholders monitor sales performance through interactive charts and KPI cards.

## 🚀 Getting Started

### Prerequisites

- [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) (latest version recommended)
- Windows OS (Power BI Desktop is Windows-only)

### Opening the Report

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/Amazon_Sales_Report_Dashboard.git
   ```

2. Open **Power BI Desktop**.

3. Go to **File → Open report → Browse this device**.

4. Select `Amazon_Sales_Report_Dashboard.pbix`.

5. If prompted to refresh data, connect to your data source or dismiss to view the existing data model.

## 🔄 Data Refresh

> **Note:** This `.pbix` file was created in the **Power BI Cloud** environment. If the data source requires credentials or a live connection, you will need to configure the data source settings in Power BI Desktop under:  
> `Home → Transform data → Data source settings`
