## [1.0.0] - 2026-04-19

### Added
- Initial release of Amazon Sales Report Dashboard
- Single-page Power BI report with 8 visuals
- KPI Card for summary metrics
- Slicer for interactive filtering
- Bar Chart for categorical comparisons
- Pie Chart for sales distribution
- Line Chart for trend analysis
- Column Chart for period-based comparisons
- Line & Clustered Column Combo Chart for combined analysis
- Amazon branding image header
- Report created via Power BI Cloud (Release 2026.03)
